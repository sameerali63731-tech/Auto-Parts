# PowerSource Auto Repair

A responsive landing page for an engines & transmissions auto-repair shop, built with **Next.js 14** (App Router) and TypeScript. The design is a faithful replication of the provided Figma mockup, using flat SVG icons (no 3D icons) and custom CSS.

## Sections

- Sticky header with desktop nav + mobile hamburger menu
- Hero with engine/transmission imagery and warranty card
- Feature strip (Quality / Best Prices / Wide Selection / Expert Support)
- Top selling inventory cards
- 3-step process
- Customer testimonials
- Domestic & import brand specialists
- "Why Choose Us" stats
- Contact CTA + appointment booking form
- Footer

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build
npm start
```

## Tech

- Next.js 14 (App Router)
- React 18 + TypeScript
- Custom CSS (`app/globals.css`), fully responsive across desktop, tablet, and mobile
- Inline flat SVG icon set (`app/components/Icons.tsx`)
